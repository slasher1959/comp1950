Favourite Web Developer Tools

Notepad++

An easy to use editor that can handle most any code syntax used in web development

HTML Kit

An easy to use, entry level tool that has excellent support for HTML

Eclipse

Probably the most popular, freely available, IDE in use for web and enterprise type development today.  Supports most any code syntax, including enterprise type languages such as Java.

Netbeans

Probably the second most popular, freely available, IDE in use for web and enterprise type development today.  Competes directly with Eclipse.   Has similar functionality to Eclipse. 

Microsoft WebMatrix

Is a free, cloud-connected application for Windows. WebMatrix enables developers to build websites using built-in templates or popular open-source applications, with full support for many of the most popular used technologies. Provides web developers with coding, customization, and publishing capabilities all in one place.

Microsoft Expression Web

Microsoft Expression Web, code-named Quartz, is a feature rich web development editor and design tool product.    It is available free of charge from Microsoft.  Support and development has been discontinued, however I love this tool and continue to use it.

XML Copy Editor

XML Copy Editor is a free XML Editor that has good features for editing XML files.   It provides a variety of XML validation options and easily checks the well formedness of XML
